1. Пользователь открывает приложение, открывается первое окно
	1.1. Пользователь кликает на кнопку "Новая заметка".
	Открыватся второе окно с пустыми полями.
	1.2. Если заметки уже есть - пользователь кликает на любую из существущих заметок. 
	Открывается второе окно с заполненными полями [для дозаполнения или просмотра].
	1.3. Пользователь отмечает в чекбоксе заметки, которые он хочет удалить и кликает на кнопку "Удалить".
	Если пользователь выбрал хоть одну заметку в чекбоксе, заметки стираются.
2. Второе окно
	2.1. Пользователь кликает на поле "Заголовок". 
	Появляется пустое поле того же размера с курсором внутри. 
	Пользователь может вводить туда свои данные.
	2.2. Пользователь кликает на поле "Содержимое". 
	Появляется пустое поле того же размера с курсором внутри. 
	Пользователь может вводить туда свои данные.
	2.3. Пользователь кликает на поле "Время". 
	Появляется пустое поле с того же размера с курсором внутри. 
	Пользователь может вводить туда время [через точку, проверять через регулярки]
	2.4. Когда поля "Заголовок" и "Время" заполнены, // появляется кнопка "ОК". // можно пока не реализовывать
	При нажатии заметка сохраняется в новом файле и закрывается второе окно.
